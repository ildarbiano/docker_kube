=========== Docker Desktop через Powershell ==========================================
powershell
Get-Service -Name "Docker Desktop Service"				# Проверьте, запущен ли Docker Desktop и Kubernetes
Get-Service | Where-Object {$_.Name -like "*docker*"}			# Посмотреть все службы Docker
Get-Service | Where-Object {$_.DisplayName -like "*Docker*"}		# найти службу по отображаемому имени
Get-Process "Docker Desktop" -IncludeUserName | Format-Table Name, Id, UserName	# Найти процесс Docker Desktop с указанием пользователя
Stop-Service -Name "Docker Desktop Service" -Force			# Остановить Docker службу
Stop-Process -Name "Docker Desktop" -Force -ErrorAction SilentlyContinue	# 
& "C:\Program Files\Docker\Docker\Docker Desktop.exe"			# Запустите Docker Desktop через исполняемый файл
Start-Process "C:\Program Files\Docker\Docker\Docker Desktop.exe"	# Запустите Docker Desktop через Start-Process
Start-Service -Name "Docker Desktop Service"				# Запустить Docker службу  
Restart-Service -Name "Docker Desktop Service" -Force			# Или одной командой

================= Установите WSL 2 - Windows Subsystem for Linux 2 ================================
WSL 2 — это вторая версия подсистемы Windows для Linux, позволяющая запускать полноценную среду Linux непосредственно в Windows, без необходимости использования виртуальной машины или двойной загрузки. Она использует технологию виртуализации для запуска собственного ядра Linux в упрощенной служебной виртуальной машине, что обеспечивает полную совместимость с системными вызовами и значительно повышает производительность файловой системы. 
	1. Установите WSL 2 
		powershell
wsl --list --online			# доступные дистрибутивы
wsl --install				# установите дистрибутив по умолчанию:
wsl --install -d Ubuntu			# Установите Ubuntu (рекомендуется)
wsl --list --verbose
		NAME      STATE           VERSION
		* Ubuntu    Running         2
wsl --shutdown				# Или остановите все WSL дистрибутивы

================= Установите Chocolatey (требует прав администратора) =============
Chocolatey — это менеджер пакетов для Windows, который позволяет устанавливать, обновлять и удалять программное обеспечение через командную строку. Он аналогичен менеджерам пакетов в Linux, таким как apt или yum, и упрощает процесс автоматической установки тысяч программ. Chocolatey основан на технологиях PowerShell и NuGet, что позволяет управлять программами с помощью простых команд, минуя ручную установку с сайтов разработчиков. 
Запустите PowerShell от имени администратора и выполните:
Set-ExecutionPolicy Bypass -Scope Process -Force
[System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072
iex ((New-Object System.Net.WebClient).DownloadString('https://community.chocolatey.org/install.ps1'))
powershell
choco install kubernetes-cli -y			# Установите последнюю версию kubectl
